About Me

Merupakan lulusan informatika dari STMIK Horizon Karawang dengan 3.00, aktif di Badan Eksekutif Mahasiswa dan komunitas Linux Horizon semasa kuliah. Telah menyelesaikan matakuliah penting, seperti : algoritma dan pemrograman dasar, struktur data, object oriented programming, system development life cicle, pemrograman web dasar, pemrograman web lanjut, pemrograman mobile, cloud computing, data mining, sistem pakar dan sistem penunjang keputusan.

Menguasai pemrograman web dengan HTML, PHP, CodeIgniter(PHP Framework), JavaScript kemudian koneksi database dengan MySql cukup mempuni dalam API, REST API, RESTful API dan Version Control System dengan GIT. Kemudian berpengalaman menggunakan bahasa pemrograman lain, seperti python, C, C++, C#, Java dan pemrograman mobile dengan Flutter. Dan Keahlian dalam Administrasi
